This API is suited to fetch and create data for the ScratchMap that can be accessed via:

https://scratchmap-iota.vercel.app/

API URL:

https://scratchmap-backend.onrender.com

Trips:

  1. GET https://scratchmap-backend.onrender.com/trips

    Delivers all trips published in Contentful with enriched data from ElephantSQL.


  2. POST https://scratchmap-backend.onrender.com/trips/:id

    Adds information about the trip :id to the ElephantSQL table "footprint". Expects a body with following information: placeDeparture String, placeArrival String, vehicle String. They are equivalent to the parameters {dest} and {typ} of the API CarbonTracer. For specification check its documentation on https://carbontracer.uni-graz.at/api-doc#W_IV.

    Example body {
    "placeDeparture": "Frankfurt-Hahn Airport",
    "placeArrival": "Corfu Airport",
    "vehicle": "flight-economy"
    }

  3. PUT https://scratchmap-backend.onrender.com/trips/updateCompensation/:id?set={true || false}

    Updates the state of the boolean field "compensated". Expects the id of the trip and a query parameter "set" with either true or false.

    Example http://localhost:8080/trips/updateCompensation/4g2oAcEVDpNyJsX8scqak3?set=true

Experiences:

  1. GET https://scratchmap-backend.onrender.com/experiences

    Delivers all experiences published in Contentful with enriched data from ElephantSQL.


  2. POST https://scratchmap-backend.onrender.com/experiences/images/update

    Adds information about the experience to the ElephantSQL tables "colours" and "exif". Expects body with following information: id: id of the experience that has to be updated. url: url of the image that the colour and exif data should be extracted from. updateColour: boolean, if the colour needs to be extracted. updateExif: boolean, if the exif data needs to be extracted.

    Example body {
    "experiencesNeedUpdate": [
    {
    "id": "gewcawecrwe",
    "url": "https://images.ctfassets.net/q0wjbbqqoctx/21trwSvi3IE9aW5ezIfXtw/4955661ebe310cde287ff6ff00468dfa/IMG_3025.JPG",
    "updateColour": true,
    "updateExif": true
    }
    ]
    }
